## What is Type Inferance?

`Type Inferance` বলতে বুঝায় যখন আপনি কোণ ভেরিয়েবল এর টাইপ নিজে না বলে দেন, তখন টাইপ্সক্রিপ্ট নিজে থেকেই ভেরিয়েবল এর ভ্যালু দেখে বুঝে নেয় সেটা কোন টাইপ। এটাকেই মূলত `Type Inferance` বলে।

যেমনঃ
```tsx
    const greet = "Hello"
```
এখানে আমরা টাইপ নিজে থেকে বলে দেইনি। কিন্তু এখানে  ভ্যালু দেখেই টাইপ্সক্রিপ্ট বুঝে নিবে যে  এটা `string` টাইপ এর ডাটা।

## What is Type Annotation?

`Type Annotation` বলতে বুঝায় যখন আমরা নিজে থেকে কোন টাইপ সেট করে দেই। 
যেমনঃ
```tsx
    const someThing: string = "Hello"
```

এখানে আমরা টাইপ নিজে থেকে বলে দিয়েছি যে এটার ভ্যালু টাইপ হবে `string`। তাই এখানে টাইপ্সক্রিপ্ট সুধু `string` টাইপ কেই এলাউ করবে ।
